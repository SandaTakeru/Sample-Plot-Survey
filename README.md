# Sample-Plot-Survey
This plugin compares and evaluates the values ​​of raster layers for each area. It can be used for land consolidation and sample plot surveys in forests.<br/>
このプラグインは、森林における標準地調査の候補地選定を支援します。衛星画像を元に標準的な林相のエリアを抽出します。襲用可否の判断にも活用できます。<br/>
This plugin is experimental version!　これは試験中のプラグインです。計算結果と現地調査結果の差異について、統計的解析・評価を行った上で正式リリースを予定しています。
